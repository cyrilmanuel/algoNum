recupérer le pojet 
1. git clone https://isic-lan.he-arc.ch/git/1415-appweb-g3.git


Travaille sur le projet, ensuite

1. git add -A
2. git commit -m "Kesske t'as fait"
3. git pull 	(pour etre sûr que t'as les dernières modif')
4. git push


git status:
qui t'indique l'état de tes fichiers (ajouté/modifiés/supprimés)


mise en place du tag :
git tag -fa version-2.5 -m "version-2.5" 75f5775e   (75f5775e = id commit)

ensuite pour valider :
git push -f --tags

vue des branches existante et branche active :
git branch			(la croix indique la branche sur la quelle on se trouve)

créer une branche :
git checkout -b "NOMDELABRANCHE"

Changer de branche :
git checkout NOMDELABRANCHE

fusionner les branches:
1. git checkout NOMDELABRANCHEQUIRECOITLEMERGE
2. git merge NOMDELABRANCHEQUISERAMERGER

(exemple on veut metre la branche issue 53 dans master. 
soit remplacer la version master par celle de issue 53)
1. git checkout master
2. git merge issue53

pour les commits sur branches :

git push origine NOMDELABRANCHEENCOURS
